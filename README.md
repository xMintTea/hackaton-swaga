**Cyberskill** — прототип образовательной платформы, созданный для хакатона «ИТ-старт» на площадке «Инновационная IT-точка» во ВСГУТУ с 16 по 18 сентября 2025 года.

Задача была — разработать платформу, включающую один из трёх модулей: геймификация, социальное взаимодействие или персональный мотиватор. Мы сосредоточились на геймификации, а остальное добавляли по мере сил. В итоге задумка выросла в идею локального комьюнити — пространства не только для учёбы, но и для самовыражения.

Мы — студенты СПО, самые молодые участники, — заняли с этим проектом второе место.

---

## Текущее состояние проекта
*Обновлено: 13 октября 2025*

Проект находится в послеконкурсной стадии. Из-за ограниченного времени нам пришлось идти на компромиссы: какие-то функции остались нереализованными, а код кое-где получился сложным для чтения и развития.

**Что уже работает**:
- Регистрация и авторизация учеников
- Тест для подбора курса: любой пользователь может пройти его и получить рекомендации
- Все основные страницы и навигация между ними
- Простая админка для редактирования данных в БД

## Планы по развитию

Мы хотим довести проект до полноценной рабочей версии. План такой:

1. **Разобраться с техническим долгом** — пересмотреть решения, принятые в спешке, улучшить читаемость и структуру кода.
2. **Улучшить интерфейс** — доработать визуал, отполировать готовые страницы.
3. **Добавить нереализованные функции** — внедрить то, что не успели в рамках хакатона.
4. **Протестировать и собрать обратную связь** — оценить, как система работает в реальных условиях.
5. **Выпустить первую стабильную версию** — закончить с демо и подготовить проект к дальнейшему развитию.
6. **Дальнейшие улучшения** — развивать платформу на основе отзывов и новых идей.
