Тестирование навыков

Страница в разработке

Здесь будет система тестирования для определения уровня знаний по различным направлениям программирования.

Комментарий для бэкендера: необходимо реализовать систему вопросов с различными типами ответов, алгоритм подсчета баллов, систему рекомендаций на основе результатов и интеграцию с лидербордом.

Планируемый функционал:

  • Разделение тестов по категориям (Python, JavaScript, Java и др.)
  • Различные типы вопросов: одиночный выбор, множественный выбор, ввод кода
  • Система подсчета очков и определения уровня
  • Рекомендации по курсам на основе результатов
  • Добавление результатов в лидерборд
  • Система достижений и наград
